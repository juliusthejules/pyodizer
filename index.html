Pyodizer™

A Python-based privacy and security configuration manager

What is Pyodizer?

Pyodizer is a Python script designed to enhance privacy and security by modifying key configurations on your device. It anonymizes device information, blocks external access, and enhances encryption, making your setup more secure.

Key Features

  • Device Anonymization: Randomizes the MAC address to prevent device tracking.
  • IP Address Management: Blocks external IP access by setting IPv4 and IPv6 addresses to localhost.
  • Firewall Rules: Configures firewall rules to block unwanted traffic such as HTTP, HTTPS, and ICMP.
  • DNS & Hostname Configuration: Ensures DNS and hostname are set to localhost for enhanced privacy.
  • Telemetry Disabled: Disables telemetry to prevent data collection.
  • Encryption: Enables AES-256 encryption to secure communications and data.
